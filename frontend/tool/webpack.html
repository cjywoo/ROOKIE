webpack工具

webpack的核心概念

  • 入口: entry
  • 输出: output
  • Loader: module
  • 插件: plugins
  • 模式/兼容性: 利用mode属性，可以切换development和production

loader和插件的区别： loader去处理webpack不能处理的文件(即除了js文件)，转化为可以处理的文件； 插件的范围更广，从打包优化到压缩。

如何调试

  • 利用 node --inspect-brk指令进行调试

    # 访问 chrome://inspect/#devices
