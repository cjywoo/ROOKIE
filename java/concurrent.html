这里是高并发的相关知识

1.高并发

通过设计来保证系统具有同时并行处理多个请求的能力

2. 并发与高并发

  • 并发：多个线程同时操作相同的资源，保证线程安全，合理使用资源
  • 高并发：服务同时处理很多请求，提高程序性能
